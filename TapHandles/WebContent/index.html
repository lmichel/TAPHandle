TAPHandle Node Selector @ b

 1- Connect a TAP Service

  • Select the TAP service you want to connect with in the Node Selector field.
  • Type free text and follow the GAVO TAP registry suggestions.
  • Or type a full URL.
  • Double click on a service node to filter the table set.

 2- Access both data and meta-data

  • Double click on a table icon on the resource tree to see its content.
  • Click on to expend/reduce the query editor.
  • Drop a table leaf from the resource tree on to display the column description.
